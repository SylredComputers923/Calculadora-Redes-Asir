# Scripts para Gestión de Redes  

Este repositorio contiene varios scripts útiles relacionados con la gestión y configuración de redes. Todos los scripts están escritos con el propósito de facilitar tareas comunes como cálculo de subredes, sumarización de rutas y generación de comandos para configuraciones en Packet Tracer.

## Scripts Incluidos  

### 1. **Calculadora IP**  
   - **Descripción**: Este script calcula los rangos de IP, máscara de red y broadcast a partir de una dirección IP inicial y una máscara de red.  
   - **Uso**:  
     - Ingresa una dirección IP y el script calculará todos los detalles de la subred correspondiente.  

### 2. **Sumarización de Rutas**  
   - **Descripción**: Ayuda a sumarizar múltiples redes en una única entrada de ruta más eficiente.  
   - **Uso**:  
     - Introduce un rango de redes y el script generará la ruta sumarizada más adecuada.  

### 3. **Generador de Comandos para Packet Tracer**  
   - **Descripción**: Este script crea automáticamente los comandos necesarios para configurar subredes basándose en una IP inicial y una IP final.  
   - **Uso**:  
     - Introduce la IP inicial, la IP final y la cantidad de subredes requeridas. El script generará comandos listos para usar en routers y switches.
